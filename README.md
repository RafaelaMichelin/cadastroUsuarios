<h1 align="center"> :bust_in_silhouette: Sistema cadastro de usuários </h1>
<h2> :pushpin: Descrição do projeto:</h2>
<p>Este projeto é um sistema web de cadastro de usuários desenvolvido com Spring Boot, seguindo uma arquitetura em Model, Controller, Service e Repository.

O sistema permite cadastrar, listar, editar e excluir usuários, contendo validação de dados e persistência no MySQL, utilizando também o Bootstrap, jQuery e DataTables para o front-end.

<h2> :space_invader: Tecnologias utilizadas:</h2>

`- Java 21`

`- Spring boot 3.x`

`- Spring Data JPA`

`- MySQL`

`- Thymeleaf`

`- Bootstrap`

`- Jquery e Datatables`
