<h1 align="center"> :bust_in_silhouette: Sistema cadastro de usuários </h1>
<h2> :pushpin: Descrição do projeto</h2>

<p> Este projeto é uma aplicação web para cadastro e gerenciamento de usuários, desenvolvido como parte do processo de aprendizado
de desenvolvimento web Java e Spring Boot.</p>

<h2> :brain: Aprendizados e objetivos </h2>
<p>
O principal objetivo deste projeto foi colocar em prática conceitos fundamentais do
desenvolvimento web com Java, utilizando o Spring Boot. Durante o desenvolvimento,
foram aplicados conceitos de arquitetura em camadas, persistência de dados com
JPA/Hibernate, integração com banco de dados e criação de páginas dinâmicas com
Thymeleaf e Jquery, permitindo compreender o fluxo completo de uma aplicação web, do back-end
ao front-end.
</p>

<h2> :space_invader: Tecnologias utilizadas</h2>

`- Java 21`

`- Spring Boot`

`- Spring Data JPA`

`- MySQL`

`- Thymeleaf`

`- Bootstrap`

`- JQuery e Datatables`

<h2> :hammer_and_pick: Funcionalidades</h2>

 * **Cadastro de Usuário:** Validação e persistência via JPA.

 * **Listagem Dinâmica:** Tabela interativa com busca e ordenação via DataTables.

 * **Edição de Usuários:** Recuperação de dados para atualização de registros existentes.

 * **Exclusão de Usuários:** Remoção de usuários do sistema</p>
